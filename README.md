# SFWRENG 2AA4: Settlers of Catan

## Project Overview
[cite_start]This project is a digital implementation of the **Settlers of Catan** board game. [cite_start]It is developed for **McMaster University's SFWRENG 2AA4 (Software Design I)** course using an iterative-incremental lifecycle model. 

## Development Status
The system progresses through three major increments:
1. [cite_start]**Simple Simulator:** Logic for game rules and action traces.
2. [cite_start]**Human Gameplay:** Interactive UI and regression testing.
3. [cite_start]**Machine Intelligence:** Rule-based AI and design patterns.

## Quality & Management
* [cite_start]**Design:** Heavily focused on **SOLID** principles and domain modeling.
* [cite_start]**Workflow:** Managed via a **Kanban board** with all commits linked to specific work items.
* [cite_start]**Quality Gate:** Code quality is monitored via **SonarQube**.
